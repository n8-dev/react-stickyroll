Contribute to Stickyroll

Please read our contributing guide

git clone https://github.com/stickyroll/react-stickyroll.git
cd react-stickyroll
yarn
yarn dev

# Test your changes
yarn test
# scoped tests
yarn lerna run test --scope @stickyroll/<package_name>
